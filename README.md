# TECYES

Sitio web de **TECYES S.A.S.** — soluciones integrales de tecnología, limpieza y consultoría para empresas industriales y conjuntos residenciales en Colombia.

Página web estática (HTML + CSS, fuente Inter vía Google Fonts). El formulario de contacto usa [Formspree](https://formspree.io/) y los botones de contacto abren WhatsApp.

## Publicación

Publicado con **GitHub Pages** desde la rama `main`.

> Nota: el `index.html` incluye una etiqueta `<link rel="canonical">` y metadatos Open Graph que apuntan a `https://www.tecyes.com.co`. Si publicas con un dominio distinto (por ejemplo la URL de GitHub Pages), actualiza esas URLs.

## Desarrollo

No requiere instalación ni build. Abre `index.html` en el navegador.
